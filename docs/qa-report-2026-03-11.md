## 12. 紧急修复补丁（快捷按钮不可点击）

问题：
- 快捷操作条（聊天/喂食/抚摸/清洁）在部分时机可见但不可点击。

根因：
- 透明窗口处于 `setIgnoreMouseEvents(true)` 状态时，快捷操作条未及时强制切回可点击状态。

修复：
- `Caterpillar.updateIgnoreMouseEvents()` 增加 `force` 参数，允许强制同步主进程穿透状态。
- `QuickActions` 在显示、进入、按下、点击时强制执行 `setIgnoreMouseEvents(false)`。
- 按钮点击增加 `stopPropagation + preventDefault`，避免全局点击链路干扰。

修复后验证：
- `npm run check`：通过
- `npm run build:mac:local`：通过
- 新 DMG SHA256：`7da8852312480ed46ae226e6cf728c9975698600f206cc293104c9960f0844d0`

## 13. 紧急修复补丁（移向按钮时快捷条消失）

问题：
- 点击宠物后快捷条出现，但鼠标移动到按钮区域时快捷条先消失，导致无法操作。

根因：
- `pet:hover-change` 在鼠标离开宠物本体时立即触发隐藏；宠物和按钮之间存在移动过渡时间，触发了误隐藏。

修复：
- `QuickActions` 增加 `petHovered / actionHovered` 状态。
- 隐藏策略改为“延迟 220ms + 条件隐藏”，鼠标进入快捷条区域会取消隐藏。
- 面板级隐藏（设置/角色管理打开）仍保留强制立即隐藏。

修复后验证：
- `npm run check`：通过
